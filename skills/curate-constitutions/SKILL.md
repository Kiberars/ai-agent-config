# SKILL: curate-constitutions

## Purpose
Autonomously fetch, score, and archive the best `PROJECT_CONSTITUTION.md` examples from the `Kiberars/ai-agent-config` GitHub repository into a local library file. Never modifies any file in the source repository.

## Trigger
- Explicit: `/curate-constitutions`
- Keyword: "отобрать конституции", "обновить библиотеку конституций", "найди лучшие примеры constitution"
- Scheduled: at the start of any new project, before filling in `PROJECT_CONSTITUTION.md`

---

## GitHub Source

**Repository:** `https://github.com/Kiberars/ai-agent-config`
**Scan paths (in priority order):**
1. `examples/*/PROJECT_CONSTITUTION.md` — project-specific filled examples
2. `examples/*/*.md` — any constitution-like file inside example projects
3. `templates/PROJECT_CONSTITUTION.template.md` — canonical template (reference only, not scored)

**GitHub API base:** `https://api.github.com/repos/Kiberars/ai-agent-config`

**Fetch sequence:**
```
GET /contents/examples                    → list example projects
GET /contents/examples/{project}/         → list files per project
GET /contents/examples/{project}/PROJECT_CONSTITUTION.md → raw content (field: content, base64)
```

Decode base64 content before scoring:
```python
import base64, json, urllib.request

def fetch_json(url):
    req = urllib.request.Request(url, headers={"User-Agent": "PromptGen-Curator/1.0"})
    with urllib.request.urlopen(req) as r:
        return json.loads(r.read())

def fetch_file(api_url):
    data = fetch_json(api_url)
    return base64.b64decode(data["content"]).decode("utf-8")
```

## Scoring Rubric

Score each fetched constitution on 6 dimensions (0–2 pts each, max = 12):

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| **Completeness** | >50% placeholders (`<!-- ... -->`) remain | 25–50% placeholders | <25% placeholders — all key sections filled |
| **Specificity** | Vague goals ("build a product") | Some concrete targets | Concrete domain, users, measurable requirements |
| **Tech Stack** | Empty table | Partial (≥2 rows) | Full table with `Reason` column filled |
| **Domain Model** | Missing or `Entity A / Entity B` skeleton | 2+ real entities, fields untyped | 2+ real entities with typed fields and relationships |
| **Constraints** | Empty or generic ("Do not use bad code") | 1–2 real prohibitions | 3+ explicit, actionable prohibitions |
| **AES Compliance** | No checkboxes marked | L1 marked | L2+ marked with notes |

**Threshold for inclusion in library: score ≥ 8 / 12**

---

## Scoring Function (Python)

```python
import re

def score_constitution(text: str) -> dict:
    score = 0
    notes = []

    # Completeness: count unfilled placeholders
    placeholders = len(re.findall(r'<!--.*?-->', text, re.DOTALL))
    total_sections = text.count('##')
    ratio = placeholders / max(total_sections, 1)
    if ratio < 0.25:   score += 2
    elif ratio < 0.5:  score += 1
    notes.append(f"placeholders={placeholders}")

    # Specificity: Purpose line not a template sentence
    if re.search(r'Purpose.*?:.{20,}', text) and '<!-- ' not in (re.search(r'Purpose.*', text) or type('', (), {'group': lambda s: ''})()).group():
        score += 2 if len(re.findall(r'\b(user|system|api|service|client)\b', text, re.I)) > 3 else 1

    # Tech Stack: filled rows
    stack_rows = re.findall(r'\|\s*\w+\s*\|\s*\w+\s*\|\s*\w', text)
    if len(stack_rows) >= 5:   score += 2
    elif len(stack_rows) >= 2: score += 1

    # Domain Model: real entities
    entities = re.findall(r'^[A-Z][a-zA-Z]+\n\s+- \w+: \w+', text, re.MULTILINE)
    if len(entities) >= 2:   score += 2
    elif len(entities) >= 1: score += 1

    # Constraints: Do not / Never
    prohibitions = re.findall(r'(Do not|Never|Запрещено)\s+\w+', text)
    if len(prohibitions) >= 3:   score += 2
    elif len(prohibitions) >= 1: score += 1

    # AES: checkboxes
    checked = len(re.findall(r'\[x\]', text, re.I))
    if checked >= 2:   score += 2
    elif checked >= 1: score += 1

    return {"score": score, "notes": notes}
```

## Output: Local Library File

**Path:** `examples/CONSTITUTION_LIBRARY.md`
**Read-only marker at top of file:** `<!-- AUTO-GENERATED by PromptGen /curate-constitutions — DO NOT EDIT MANUALLY -->`

File structure:
```markdown
<!-- AUTO-GENERATED by PromptGen /curate-constitutions — DO NOT EDIT MANUALLY -->
# Constitution Library
> Curated from github.com/Kiberars/ai-agent-config · Last updated: {ISO date} · {N} entries

---

## {Project Name} — Score: {N}/12
**Source:** `examples/{project}/PROJECT_CONSTITUTION.md`
**Strengths:** {comma-list of top-scoring dimensions}

{full constitution content, verbatim}

---
```

**Merge rules (never destroys existing entries):**
1. Load existing `CONSTITUTION_LIBRARY.md` if present.
2. For each fetched constitution with score ≥ 8:
   - If source path already exists in library → compare scores, keep higher-scored version.
   - If new → append to file.
3. Write updated file to `examples/CONSTITUTION_LIBRARY.md` locally.
4. Never `git add`, `git commit`, or `git push`.

---

## Rules

- Do not modify any file under `examples/*/` source paths — only read them via API.
- Do not include constitutions with score < 8 in the library.
- Do not include the template file (`PROJECT_CONSTITUTION.template.md`) as a curated example — list it as "Reference Template" separately with no score.
- Do not call the GitHub API more than once per file (cache in-memory during the session).
- Always include the source URL and fetch date in each library entry.
- If the API returns rate-limit (HTTP 403/429), wait 60 s and retry once, then abort with a clear error message.

---

## Integration with PromptGen Workflow

When the user runs `/curate-constitutions`, PromptGen:
1. Fetches the examples directory listing via GitHub API.
2. Scores all found constitutions.
3. Reports a summary table: `| Project | Score | Included? |`
4. Writes/updates `examples/CONSTITUTION_LIBRARY.md`.
5. Outputs: "Library updated: {N} entries. Best example: {project} ({score}/12)."

When filling a new `PROJECT_CONSTITUTION.md`, PromptGen automatically loads the top-3 highest-scored entries from `CONSTITUTION_LIBRARY.md` as few-shot references before generating.
